# RiskDue

RiskDue is a web application that helps students evaluate the risk of missing assignment deadlines based on workload and time remaining.

## Features

- Add assignments with deadlines and estimated work hours  
- Calculates required hours per day  
- Predicts risk level (Low, Medium, High)  
- Edit and delete assignments  
- Data persistence using localStorage  
- Dashboard showing total tasks, high-risk tasks, and total workload  

## How It Works

The app estimates workload using:

hours_per_day = estimated_hours / days_left

Risk levels are determined based on how demanding the workload is relative to the remaining time.
